# プロファイルの新規作成

```File``` → ```プロファイルを新規作成```を開くと、新規プロファイルのダイアログが表示されます。\
プロファイル名を入力して[OK]を押すことでプロファイルが作成されます。

{{<hint info>}}
※各設定は直前まで表示されていた内容を引き継ぎます。
{{</hint>}}

## プロファイルの保存について
- 設定変更後、💾を押すと上書き保存されます。
- プロファイル名を書き換えて💾を押すとその名称のプロファイルが生成され、そのプロファイルに設定が保存されます。書き換え前のプロファイルには設定の変更は反映されません。
