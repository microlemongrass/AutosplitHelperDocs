# このソフトって何？

AutoSplit Helper（以下ASH）とは…

・RTAのラップを自動で取るソフトです。  
・画像認識を利用しています。

　→TVゲーム等、ゲーム内メモリを利用できないゲームでも自動でラップを取ることが出来ます。

（画像認識の仕様上、取れない場合（場面）もあります）
